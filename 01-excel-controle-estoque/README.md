# Projeto 01 — Controle de Estoque com Excel

## Sobre o projeto

Este projeto simula um cenário empresarial de controle de estoque para uma empresa fictícia chamada LLC Electronics.

O objetivo é construir uma planilha profissional em Excel capaz de acompanhar entradas, saídas, saldo atual, valor financeiro em estoque e status dos produtos.

## Objetivos da análise

- Controlar entradas e saídas de produtos.
- Calcular saldo atual por SKU.
- Calcular valor financeiro em estoque.
- Classificar produtos por status.
- Criar um painel visual para tomada de decisão.

## Ferramentas utilizadas

- Microsoft Excel
- Tabelas estruturadas
- Fórmulas
- Tabelas dinâmicas
- Gráficos dinâmicos
- Git e GitHub
